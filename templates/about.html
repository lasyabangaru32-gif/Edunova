ℹ️ About EduNova

🎓 Project Overview

EduNova is a Student Management System developed using Flask, MySQL, HTML, CSS, and JavaScript. The system helps educational institutions manage students, courses, assignments, and academic records efficiently through a modern dashboard interface.

🚀 Features

  • Student Management
  • Course Management
  • Assignment Tracking
  • Dashboard Analytics
  • Search Functionality
  • Dark Mode
  • Responsive Design

💻 Technologies Used

  • Python
  • Flask Framework
  • MySQL Database
  • HTML5
  • CSS3
  • JavaScript

🌟 Vision

EduNova aims to provide a simple, efficient, and user-friendly platform for educational management while offering a modern and professional user experience.
